# Jannat's Portfolio

A cinematic, interactive, and visually stunning portfolio website designed to showcase creativity and personality. This project leverages modern web technologies and animation libraries to create an immersive user experience.

## âœ¨ Features

- **Cinematic Intro Sequence**: A captivating visual introduction featuring a dynamic grid of images that sets the mood.
- **Interactive "About Me" Section**: A storytelling approach to the bio section, broken down into thematic blocks:
    - *Lost in Thought* (Literary interests)
    - *Quietly Unstoppable* (Personal philosophy)
    - *Simple. Real. Honest.* (Values)
    - *Strength in Motion* (Fitness and discipline)
    - *Where She Breathes Best* (Nature connection)
- **Dynamic Gallery**: A multi-layered, parallax-style image gallery that responds to user scrolling.
- **Smooth Animations**: Powered by **GSAP (GreenSock Animation Platform)** and **ScrollTrigger** for seamless transitions and scroll-based effects.
- **Responsive Design**: Optimized for various screen sizes.
